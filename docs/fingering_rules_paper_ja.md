# データ駆動型ギター運指：人間の弦選択に関する統計的法則、生体力学モデル、ニューラル予測

## 概要 (Abstract)

ギターの弦割り当て問題 — 与えられたピッチに対して人間のギタリストがどの弦・フレットを使用するかを予測する問題 — に関する包括的研究を提示する。3つの相補的アプローチ — (1) 人間選好マップ付きViterbi動的計画法、(2) 生体力学的制約付きCNNスペクトル弦分類器、(3) Transformerベース記号予測モデル — を統合し、人間の運指行動の全体像を確立した。

**8,096,865ノート**（4,238曲のクラウドソースタブ譜）からの学習と、**56,716ノート**（ヘキサフォニックピックアップ録音）による検証に基づく主要成果：

- 記号予測精度 **97.2%**（Transformer、GProTabテストセット）
- 音声ベース分類精度 **95.9%**（CNN + Biomechanical Viterbi、同一プレイヤー）
- クロスドメイン精度 **95.2%**（記号モデル → 実演録音で検証）
- 全予測の **99.7%が±1弦以内**
- 人間の弦選択の95%を支配する **5つの定量的法則**を特定

---

## 1. はじめに

### 1.1 問題定義

音源から検出されたMIDIピッチ列に対し、各ピッチをどの(弦, フレット)ペアに割り当てれば自然で弾きやすいタブ譜が生成できるかを決定する問題。

ギターの多くのピッチは複数のポジションで演奏可能。例：MIDI 60 (C4)は
- 2弦1フレット、3弦5フレット、4弦10フレット、5弦15フレットで演奏可能

人間は特定のポジションを一貫して好むが、この選好は**単純なフレット最小化ヒューリスティックでは捉えられない**。

### 1.2 研究課題

1. ピッチ文脈のみから人間の弦選択をどの程度予測できるか？
2. 弦選択を支配する要因は何か？
3. クラウドソースのパターンはプロの演奏にどの程度汎化するか？
4. 生体力学的制約と学習された統計パターンの役割分担は？

### 1.3 データソース

| データセット | ソース | ノート数 | 弦検証方法 | 役割 |
|---|---|---|---|---|
| **GProTab** | クラウドソースタブ譜 | 8,096,865 | 人手作成タブ記譜 | Transformer学習 |
| **GuitarSet** | ヘキサフォニックPU録音 | 56,716 (360トラック) | 物理的弦振動 | CNN学習 + クロスドメイン検証 |
| **IDMT-SMT-V2** | エレキギター | 5,767 | 人手XMLアノテーション | 選好マップ |
| **GOAT Dataset** | エレキギター | 1,017 | GuitarProパース | 選好マップ |

## 2. アプローチ1：人間選好マップ付きViterbi DP

### 2.1 コスト関数

```
Total Cost = Σ [Emission(i) + Transition(i, i-1)]

Emission(i) = w_fret_height × fret + w_human_pref_bonus × human_probability(pitch, string, fret) + ...
Transition(i, i-1) = w_movement × |Δfret| + w_position_shift × shift_penalty + ...
```

### 2.2 弦番号規約バグの発見

**致命的発見：** データソース間で3種の弦番号規約が混在。初期実験で見かけ上4.7%の精度を示した原因。`map_s = 7 - s` の変換で修正。

### 2.3 結果

| 構成 | 精度 |
|---|---|
| Viterbi DP（ピッチのみ） | 52.8% |
| + 人間選好 (w=-15) | 59.5% |
| + Path Difference Learning | 61.68% |

**結論：** ピッチのみのViterbiは最大~62%。理論的上限は~70%（最頻弦戦略）。さらなる改善には音声特徴量が必須。

## 3. アプローチ2：CNNスペクトル弦分類器

### 3.1 設計

| 項目 | 値 |
|---|---|
| 入力 | CQTパッチ (84bins × 11frames) + MIDIピッチ |
| アーキテクチャ | 3層CNN (32→64→128ch) + FC |
| 学習データ | GuitarSet 360トラック (61,885サンプル) |
| **Val精度** | **92.66%** |

### 3.2 同一プレイヤー vs LOPO評価

| 評価 | 精度 |
|---|---|
| 同一プレイヤー（ランダム分割） | **93.3%** |
| **LOPO（Leave-One-Player-Out）** | **80.4%** |

12.9%のギャップ → プレイヤー固有特性への過学習を確認。

### 3.3 CNNエラーパターン分析

| パターン | 件数 | 解釈 |
|---|---|---|
| S2→S1 | 300 (29.9%) | 人間はB弦ハイフレット、CNNはE弦ローフレットを選択 |
| S3→S4 | 220 (21.9%) | 人間はG弦、CNNはD弦を選択 |

**根本原因：ポジションプレイ vs 開放弦バイアス。** CNNは薄い弦を60.7%の確率で誤選択。

## 4. 生体力学的制約モデル

### 4.1 人間の手の解剖学

**指の順序制約（絶対、違反不可能）：**
```
fret(人差し指) ≤ fret(中指) ≤ fret(薬指) ≤ fret(小指)
```

**スパン制限（成人の典型的な手）:**

| 指ペア | 最大スパン |
|---|---|
| 人差し指-中指 | 3-4フレット |
| 人差し指-薬指 | 4-5フレット |
| 人差し指-小指 | 4-6フレット |

**腱結合（enslaving）：** 薬指の動きは中指・小指に不随意的に影響する。

### 4.2 Biomechanical Viterbi

状態空間に指割り当て(1-4)を追加し、生体力学的遷移コストを導入。

### 4.3 結果

| 構成 | 全体 | S2 | S6 |
|---|---|---|---|
| CNN単体 | 92.9% | 87.2% | 90.0% |
| **+ Bio w_pos=0.5** | **95.8%** | **93.8%** | 84.0% |
| + 開放弦ボーナス | **95.9%** | 94.1% | 83.2% |

S2(B弦): 87.2% → **93.8%** (+6.6%) — #1エラーパターンを大幅修正。

### 4.4 LOPO検証

| プレイヤー | CNN LOPO | Bio LOPO | 改善 |
|---|---|---|---|
| **全体** | **75.6%** | **80.8%** | **+5.2%** |

生体力学的制約はCNNの確信度が低い場合（未知プレイヤー）にこそ効果が大きい。

## 6. 人間のギター運指5法則

### 法則1：ターゲットピッチ支配

**ターゲットピッチだけで弦選択の69%が決定される。**

| 除去した特徴量 | 精度 | 低下 |
|---|---|---|
| ベースライン（全特徴量） | 96.84% | — |
| **ターゲットピッチ** | **27.80%** | **-69.04%** |
| **文脈弦番号** | **43.18%** | **-53.66%** |
| 文脈弦番号（シャッフル） | 65.22% | -31.62% |
| 直近4ノートのみ | 93.09% | -3.75% |
| 文脈フレット | 94.79% | -2.05% |
| 文脈音程間隔 | 95.34% | -1.50% |
| 文脈音価 | 96.07% | -0.77% |

**ピッチ帯域→弦の対応（50万サンプルから）：**

| ピッチ帯域 | 上位3弦 |
|---|---|
| 低域 (<E2) | 6弦=82%, 5弦=15%, 4弦=2% |
| 中低域 (E2-E3) | 5弦=69%, 4弦=16%, 6弦=15% |
| 中域 (E3-B3) | 4弦=49%, 3弦=26%, 5弦=17% |
| 中高域 (C4-F#4) | 2弦=39%, 3弦=35%, 1弦=17% |
| 高域 (≥G4) | 1弦=55%, 2弦=31%, 3弦=12% |

### 法則2：順序記憶（約4音）

Attention重み分析：直近4ポジションに注意の**38.4%**が集中。ポジションあたり**2.5倍**高い注意量。

### 法則3：ピッチ近接性は弦を保持（3半音境界）

| ピッチ間隔 | 同弦 | 隣接弦 | 遠距離弦 |
|---|---|---|---|
| **0（同音）** | **96.6%** | 3.1% | 0.3% |
| **2（全音）** | **63.0%** | 35.2% | 1.8% |
| 3（短3度） | 18.7% | 80.1% | 1.1% |
| 5（完全4度） | 2.6% | 96.0% | 1.5% |
| 7（完全5度） | 0.8% | 91.9% | 7.2% |
| 12（オクターブ） | 0.7% | 8.9% | 90.4% |

**3半音が臨界点：** 3半音未満→同弦維持が優勢、3半音以上→隣接弦遷移が優勢。

### 法則4：ポジション固執（79%が2フレット以内）

| フレット移動量 | 割合 |
|---|---|
| 停留 (0-2フレット) | **79.0%** |
| 小移動 (3-5フレット) | 16.4% |
| 大移動 (6+フレット) | 4.6% |

### 法則5：弦交替優位（遷移の92%）

| 同弦連続数 | 頻度 |
|---|---|
| 1音（即座に弦変更） | **92.2%** |
| 2音 | 3.6% |
| 3音 | 1.7% |
| 4音以上 | 2.5% |

## 9. 結論

人間のギター弦選択は、全決定の**95%**を説明する5つの定量的法則に従う：

1. **ピッチ帯域**が弦を決定（分散の69%）
2. **直近~4音の順序文脈**が選択を精緻化（弦履歴から54%）
3. **ピッチ近接性**（3半音未満）が同弦保持を強く予測
4. **ポジション固執**（79%が2フレット以内）が手の移動を最小化
5. **弦交替**がデフォルト（遷移の92%）

残り5%は隣接弦間の個人的選好 — 予測タスクの基本的限界。

**主要貢献：**
- 人間の運指法則の初の大規模定量化（800万ノート）
- クロスモーダル検証（記号予測 vs ヘキサフォニックGround Truth）
- 生体力学的制約はCNNの確信度が低い場合にこそ価値が高い（LOPO: +5.2%）
- ピッチのみアプローチの理論的上限は~70%

## 10. 先行研究との比較

### 10.1 タスクA：記号ベース（MIDI→タブ譜）

| 手法 | 年 | 学会 | データ規模 | 評価方法 | 結果 |
|---|---|---|---|---|---|
| Sayegh | 1989 | CMJ | ルールベース | 主観評価 | ベースライン |
| Radicioni & Lombardo | 2004 | ICMC | ルールベース | 専門家評価 | DP改善 |
| Hori & Sagayama (Minimax Viterbi) | 2016 | SMC | ルールベース | 主観 + DP比較 | 弾きやすさ向上 |
| Riley et al. (MIDI-to-Tab) | 2024 | **ISMIR** | DadaGP 25K曲 | ユーザスタディ（**定量値なし**） | **DPを有意に上回る** |
| **本研究 (Transformer V3)** | **2025** | — | **GProTab 4,238曲 (810万ノート)** | **定量的弦一致率** | **97.2% (テスト) / 95.2% (クロスドメイン)** |

> ISMIR 2024の「MIDI-to-Tab」（Riley et al.）は最も近い先行研究だが、**定量的精度指標を報告していない**（ユーザスタディのみ）。本研究は記号ベースギター運指予測の**初の大規模定量評価**を提供する。

### 10.2 タスクB：音声→タブ譜（弦推定）

| 手法 | 年 | 学会 | データ | 評価方法 | 指標 | 結果 |
|---|---|---|---|---|---|---|
| TabCNN (Wiggins & Kim) | 2019 | ISMIR | GuitarSet | 6-fold CV（プレイヤー混在） | TDR | **89.9%** |
| GAPS baseline | 2024 | arXiv | GAPS + GuitarSet | 教師あり | F1 | 直接比較不可 |
| **本研究 (CNN + Bio Viterbi)** | **2025** | — | **GuitarSet** | **同一プレイヤー** | **弦一致率** | **95.9%** |
| **本研究 (CNN + Bio Viterbi)** | **2025** | — | **GuitarSet** | **LOPO（未知プレイヤー）** | **弦一致率** | **80.8%** |

**指標の違い：**
- **TDR（Tablature Disambiguation Rate）：** 正しく検出されたピッチのうち正しい(弦, フレット)に割り当てられた割合。ピッチ検出が前提。TabCNN使用。
- **弦一致率（本研究）：** Ground Truthピッチを使用し、弦割り当てのみを評価。TDRよりやや寛容。

指標の違いにもかかわらず、同一プレイヤー精度（95.9%）はTabCNNのTDR（89.9%）を**+6.0ポイント**上回る。

### 10.3 本研究の独自貢献

| 貢献 | 先行研究 | 本研究 |
|---|---|---|
| 運指法則の定量化 | 大規模なものなし | **810万ノートから5法則** |
| クロスモーダル検証 | 音声 OR 記号の片方 | **両方 + 相互検証** |
| LOPO誠実報告 | ほとんど報告されない | **80.8% + プレイヤー別内訳** |
| 生体力学 + ML融合 | 文献では分離 | **Viterbi + CNN + 解剖学の統合** |
| 弦番号規約バグ | 未議論 | **3規約の特定と修正を文書化** |
| 理論的上限分析 | 非形式的 | **ピッチのみ上限70%を定量化** |

```
GProTab 4,238 GP5 → PyGuitarPro解析 → 810万ノートタプル → 曲単位分割 → 16ノート窓 → Transformer V3学習
GuitarSet 360 JAMS → ヘキサフォニック抽出 → 56Kノート → CQT+ピッチ → CNN弦分類器学習 / クロスドメイン検証
IDMT + GOAT → 手動アノテーション → 52万選好マップ → Viterbi DPコストボーナス
```

### 12.2 3アプローチ統合

```
[音声パス]   音声波形 → CQTスペクトログラム → CNN弦分類器 → 弦確率 ─┐
[記号パス]   MIDIピッチ列 → 16ノート文脈窓 → Transformer V3 → 弦確率 ─┤→ Biomechanical Viterbi DP → 最終(弦, フレット)割り当て
[人間工学パス] 選好マップ → 発出ボーナス / 手の解剖学モデル → 遷移コスト ─┘
```
